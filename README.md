# holzi-games

Same small LLM-generated games for children.

## Games

- [Robo Lanes](https://gholzmann.github.io/holzi-games/robo-lanes/) : Don't crash into other robots
- [Robot Duell](https://gholzmann.github.io/holzi-games/duel-arena/) : Duel for two players
- [Bohrer Invaders](https://gholzmann.github.io/holzi-games/bohrer-invaders/) : Space invaders variation with rockets and bohrer
- [Cupcake Studio](https://gholzmann.github.io/holzi-games/cupcake-studio/) : Cupcake Studio
